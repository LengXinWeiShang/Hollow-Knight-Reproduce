# Hollow-Knight-Reproduce
Use Unity to reproduce the BOSS fight with Hornet.

使用Unity复刻了与大黄蜂的BOSS战。
### 简介
使用代码模拟2D物理，尽可能还原了原版小骑士的跳跃和操作手感；小姐姐的AI通过有限状态机实现，根据玩家的位置和自身在房间中的位置进行行动决策。
